LOCAL PREVIEW:
1. Unzip this folder.
2. Open terminal inside the project folder where package.json is located.
3. Run: npm install
4. Run: npm run dev
5. Open: http://localhost:3000

GITHUB / VERCEL UPLOAD:
Recommended: create a NEW GitHub repo or new branch first.

Your GitHub root must show these items directly:
- app/
- components/
- lib/
- public/
- package.json
- next.config.ts
- tsconfig.json

Do NOT upload the outer folder itself. Upload the contents inside this project folder.
